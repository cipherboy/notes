# tmux

To maximize a pane in tmux:

    <tmux-prefix> + z

To scroll a pane in tmux:

    <tmux-prefix> + [

To create a new window in tmux:

    <tmux-prefix> + c

To go to the next window in tmux:

    <tmux-prefix> + n

To balance panes into equal columns:

    <tmux-prefix> + <alt + 1>

To balance panes into equal rows:

    <tmux-prefix> + <alt + 2>
